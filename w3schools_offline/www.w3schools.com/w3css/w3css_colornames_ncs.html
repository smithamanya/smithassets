W3.CSS Colors - NCS

« Previous
Next Chapter »

The Natural Color System (NCS) is the color standard (for interior design, decorating, and painting) in Sweden, Spain, Norway and South Africa.

The standard is unique because it is based entirely on the perception of colors (how eyes naturally see red, yellow, blue, green, white, and black).

NCS does not use color names. It uses letters and numbers to describe any color.

Example: The Swedish flag is blue and yellow. With NCS it can be described like this:

Color Code Black Chroma Hue
  4055-R95B 40% 55% Red + 95% Blue
  0580-Y10R 05% 80% Yellow + 10% Red

Since NCS describes colors naturally, it can not be used directly in HTML, because HEX and RGB process colors differently.

However, since the standard uses values, it is possible to use the values to compute "look alike" HTML colors.
