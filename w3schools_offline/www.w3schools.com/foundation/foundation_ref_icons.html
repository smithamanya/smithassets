Foundation Icons

Foundation provides 283 icons that can be styled with CSS: you control the size, color and style.

An icon is created with the following syntax:

<i class="fi-name"></i>

The name part in the syntax above must be replaced with the proper name of the icon.

To use the icon, you must link to the Foundation icon's stylesheet in the <head> section of your page:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/foundicons/3.0.0/foundation-icons.css">
