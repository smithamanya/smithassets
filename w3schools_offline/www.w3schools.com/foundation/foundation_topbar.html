Top Navigation Bars

A top bar is a navigation header that is placed at the top of the page:

  • WebSiteName

  • Menu
  • Home
  • Page 1
  • Page 2
  • Sign Up
  • Login

Example

<nav class="top-bar" data-topbar>
  <ul class="title-area">
    <li class="name">
      <!-- Remove this if you don't want a title/logo -->
      <h1><a href="#">WebSiteName</a></h1>
    </li>
      <!-- Collapsible Button on small screens: remove .menu-icon to get rid of icon.
      Remove the "Menu" text if you only want to show the icon -->
    <li class="toggle-topbar menu-icon"><a href="#"><span>Menu</span></a></li>
  </ul>

  <section class="top-bar-section">
    <ul class="left">
      <li class="active"><a href="#">Home</a></li>
      <li><a href="#">Page 1</a></li>
      <li><a href="#">Page 2</a></li>
      <li><a href="#">Page 3</a></li>
    </ul>
  </section>
</nav>

<!-- Initialize Foundation JS for the collapsible button to work-->
<script>
$(document).ready(function() {
    $(document).foundation();
})
</script>

Result:

  • WebSiteName

  • Menu
  • Home
  • Page 1
  • Page 2
  • Page 3
Try it Yourself »

Example Explained

A standard topbar is created with <nav class="top-bar" data-topbar>. The .title-area class defines an area for the website's logo (must be placed inside li.name) and a "menu" button that will only show on small screens; With Foundation, a topbar can extend or collapse, depending on the screen size:

On smaller screens, the topbar takes up too much space, and should be hidden and/or shown only when it is needed. The li.toggle-topbar menu.icon class replaces the links in the topbar on small screens with a button in the top right corner. Only when the button is clicked, the topbar will be displayed.
Tip: Resize the browser window to see the effect.

The .top-bar-section defines a section for navigation links. The .left class indicates that the links should be left-aligned. The .active class lets the user know which page he/she is on, and will add a blue background color to the link.

Tip: Change the .left class to .right if you want the navigations links right-aligned:

<section class="top-bar-section">
  <ul class="right">...

Result:

  • WebSiteName

  • Menu
  • Home
  • Page 1
  • Page 2
  • Page 3

Top Bar With Dropdown

The navigation links inside the top bar can also hold dropdown menus.

Create a dropdown menu by adding the .has-dropdown class to one of the <li> elements, followed by a <ul> element with class="dropdown":

Example

<section class="top-bar-section">
  <ul class="left">
    <li class="active"><a href="#">Home</a></li>
    <li class="has-dropdown">
      <a href="#">Dropdown</a>
      <ul class="dropdown">
        <li><a href="#">First link in dropdown</a></li>
        <li><a href="#">Second link in dropdown</a></li>
        <li class="active"><a href="#">Active link in dropdown</a></li>
      </ul>
    </li>
  </ul>
</section>

Result:

  • WebSiteName

  • Menu
  • Home
  • Dropdown
    • First link in dropdown
    • Second link in dropdown
    • Active link in dropdown
Try it Yourself »

Nested/Multi-Level Dropdowns

Dropdowns can also be nested (a dropdown inside a dropdown):

Example

<section class="top-bar-section">
  <ul class="left">
    <li class="has-dropdown">
      <a href="#">Dropdown</a>
      <ul class="dropdown">
        <li><label>Level 1</label></li>
        <li><a href="#">Link</a></li>
        <li><a href="#">Link</a></li>
        <li class="has-dropdown">
          <a href="#">New dropdown</a>
          <ul class="dropdown">
            <li><label>Level 2</label></li>
            <li><a href="#">2nd level dropdown</a></li>
            <li><a href="#">2nd level dropdown</a></li>
            <li class="has-dropdown">
              <a href="#">New dropdown</a>
              <ul class="dropdown">
                <li><label>Level 3</label></li>
                <li><a href="#">3rd level dropdown</a></li>
                <li><a href="#">3rd level dropdown</a></li>
              </ul>
            </li>
          </ul>
        </li>
      </ul>
    </li>
  </ul>
</section>

Result:

  • WebSiteName

  • Menu
  • Dropdown
    • Link
    • Link
    • New dropdown
      • 2nd level dropdown
      • 2nd level dropdown
      • New dropdown
        • 3rd level dropdown
        • 3rd level dropdown

Clickable

Dropdowns in topbars are hoverable by default. Add the data-options="is_hover: false" attribute to <nav> if you want them to be clickable instead:

Example

<nav class="top-bar" data-topbar data-options="is_hover: false">

Fixed Top Bar

The topbar can also be fixed at the top of the page.

A fixed topbar stays visible in a fixed position (top) independent of the page scroll.

Wrap the topbar in <div class="fixed"> to make the navigation bar fixed at the top:

Example

<div class="fixed">
  <nav class="top-bar" data-topbar>
    ...
  </nav>
</div>

Result:

Try it Yourself »

Sticky Top Bar (Affix)

You can also wrap the top bar in <div class="sticky"> and put it anywhere within your markup. When the navigation bar hits the top of the browser window, it will act like the fixed top bar and stick to the top as the user continues to scroll:

Example

<div class="sticky">
  <nav class="top-bar" data-topbar>
    ...
  </nav>
</div>

Result:

Try it Yourself »

When using the .sticky class, the top bar will be sticky on all screen sizes by default. Add the data-options="sticky_on: small|medium|large" attribute to <nav> if you want the stickiness to behave on just small, medium or large screens:

Example

<div class="sticky">
  <!-- Only sticky on large screens -->
  <nav class="top-bar" data-topbar data-options="sticky_on: large">
    ..
   </nav>
</div>

Or pass in an array to specify the sticky behavior on multiple sizes:

Example

<div class="sticky">
  <!-- Sticky on small AND large screens (NOT medium)-->
  <nav class="top-bar" data-topbar data-options="sticky_on: [small, large]">
    ..
   </nav>
</div>
